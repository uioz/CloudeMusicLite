# CloudeMusicLite

这是一个基于Electron的失败之作.

这是我的一个很久以前的项目当时Electron的最新版本还在1.8.1,希望利用NetEaseCloudMusicApi这个项目来编写一个自己的播放器.

但是需要编写的代码数量过于庞大超出了我当时的预料,加上自身的水平不行,后来被搁置至今.

这个项目依然可以运行,实际上图片展示的只是他的未加载资源的样子,因为这个项目使用的数据接口过于老旧数据获取不到.
